# Astra 360 - AI Financial Guardian

Astra 360 is a proactive AI-driven financial guardian designed to help users manage their finances, track subscriptions, and gain deep insights from their bank statements.

## Tech Stack
- **Frontend**: Vite + React + TypeScript
- **Styling**: Tailwind CSS + Shadcn UI
- **State Management**: TanStack Query
- **Icons**: Lucide React
- **Animations**: Framer Motion

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
